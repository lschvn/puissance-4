# Puissance 4

Puissance 4 en typescript, css et html. 

## Installation

Installer les dépendances
```
bun install
```

Lancer le projet en développement
```
bun ./**/*.html
```

## Structure du projet

```
assets/
	css/
		app.css
	img/
libs/
	index.ts
public/
	index.html
	game.html
	history.html
	replay.html
main.ts
README.md
```
